# LoudMurmurs_backup_script
A downloading script for podcast LoudMurmurs - 小声喧哗.

Requirements:
  * tqdm
  * feedparser
  * requests
  * argparse

Usage:

  * `python download_LoudMurmurs.py`
    
  * `python download_LoudMurmurs.py -p path_to_save`
  
  * Or just double-click the released .exe file
